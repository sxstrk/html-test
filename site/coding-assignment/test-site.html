ANTIPASTI

[лат. anti + pastico] - i need more tezt here so the line is more visible here

1) способ, с помощью которого можно избавиться от страха, тревоги и т. п. путем создания у себя ощущения удовольствия, не зависящего от внешних обстоятельств;

2) особый метод психотерапии, когда вместо того, чтобы лечить больного, психотерапевт создает у него впечатление, что он сам является больным.
